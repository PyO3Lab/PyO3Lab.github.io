# 欢迎来到 PyO3 Lab

让我们用 PyO3 尽情玩转 Rust！

我们是一个由社区驱动的计划，依靠 PyO3 将 Rust 的安全性与速度带到 Python 扩展之中。我们的使命是：

- **为 Python 加速**，引入近乎原生的 Rust 性能
- **简化集成流程**，让 Rust crate 轻松融入 Python 生态
- **打造高品质绑定**，兼顾开发体验与可维护性
- **促成合作**，连接 Rust 与 Python 社区

### 为什么重要？

现代算力需求既讲究开发效率，也追求系统性能。通过为关键 Rust crate 精心打造 Python 绑定，我们能够：

- 获得内存安全的性能提升
- 与主流 Python 生态无缝衔接
- 打开日益壮大的 Rust crate 宝库
- 保留 Python 的易用性，同时拥抱 Rust 的可靠性
